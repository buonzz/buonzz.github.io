Timestamp to Date

This is a simple form on which you can enter a UNIX timestamp, the form then will display the human readable format of that timestamp.

Date to Timestamp

This tool will accept date and then display the corresponding timestamp

Font Embedder

A tool to generate the CSS for you to enable to embed a particular font in your web page. The benefit of embedding the font in the web page itself is you can use the font even if it is not installed in the end-user's machine.

What is my IP

A tool to retrieve the information about the IP, browser and other info your browser have.
