Study how to use git, composer and behat
Use www.timedoctor.com to prevent your team-mates from sleeping during a freelance work
Use Laravel for your new projects
Be familiar with "markdown" syntax, these are .md files commonly seen in github repos
For applications that needs real-time data processing, use Ratchet
Study Map/Reduce queries and NoSQL databases like Redis and MongoDB
Write RESTfull web services using Restler
Use either github (for open source) or bitbucket (private repos) to host your source codes
Be familiar with Twitter Bootstrap for front-end development
Encrypt and license your source codes using ioncube
