Starting a new project?
Take a look at this list and let me know what you think.

Programming Language : PHP
Framework: CodeIgniter Bonfire
Database Server: Amazon EC2
Web Server: Nginx
Database: CouchDB
JavaScript Framework: jQuery
CSS / Ui Framework: Twitter BootStrap
Authentication: Oauth 1.0
Website Font: Typekit
Debugging Tools: FirePHP
Version Control: GitHub

This is the best softwares I recommend for someone who will be building a new site/application. Of course some of this doesn't apply in all projects, but I consider these tools to be on the top of their own in terms of balanced costs, portability and speed of development.
