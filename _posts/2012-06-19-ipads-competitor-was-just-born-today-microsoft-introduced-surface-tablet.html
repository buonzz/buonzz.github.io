Few hours ago, in Los Angeles California, a bunch of people gathered for the "mystery" event that Microsoft is going to conduct.

As it was revealed, they introduced the "Surface" tablet

This tablet has the ff. specs:

  • 10.6-inch tablet
  • Wrapped in a magnesium case that Microsoft calls VaporMg
  • has magnetically-attached cover that folds down and functions as a full touchscreen keyboard
  • Has 2 versions, Windows RT ( Nvidia ARM ) and  Windows 8 Pro ( Intel Core i5)
