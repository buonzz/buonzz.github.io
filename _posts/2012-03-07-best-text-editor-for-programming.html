One of the primary tools of a Web Developer are text editor. Not just a text editor but a powerful text editor.

As a website grows, so do the complexity of the codes. In such cases, having an easy to use text editor is really a life-saver.

Notepad++ is one of the best editor I ever met

http://notepad-plus-plus.org/

It's a all purpose, light-weight program that you can use in everyday development. In the kind of job I have, wherein I usually modifies/debugs the codes of existing sites written by other programmers, I don't want to get myself in trouble in character encoding issues and getting hanged by heavy IDE like Eclipse.
