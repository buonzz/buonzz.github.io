Download WAMP in here
Download MongoDB in here

Install those 2

Add the C:\Program Files\MongoDB 2.6 Standard\bin to the system path

Create these folder in your C: drive

c:\mongodb\data
c:\mongodb\log

create a file in named mongod.cfg and put these inside that file

logpath=C:\mongodb\log\mongo.log
dbpath=C:\mongodb\data

save it on c:\mongodb\

create a file named start.bat, put this inside

mongod -config C:\mongodb\mongod.cfg --profile=1 --slowms=1
pause

save it on c:\mongodb\

to start the server, double click the start.bat
then open a new console and type mongo

when in the mongo prompt

use

to select database

db.getCollectionNames();

to get all the collections available
