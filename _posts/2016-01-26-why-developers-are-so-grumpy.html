Let me show you some example first to better illustrate and put some context. Let us start with the most grumpiest of them all -- Linux Torvalds (father of Linux ). One of the few remarkable ones are

Linux Pull Request Meltdown

This post you can see more details about the incident, wherein a contributor had submitted a pull request for some changes and unfortunately Linus find the changes not satisfying, this had caused him to utter words such as crap,shit and idiot.

Nvidia Fuck You!

Watch this video to see how Linus pulled out his middle finger during a public conference, facing the camera straight and saying "Nvidia Fuck You!"

Java is a horrible language

An interview with Linux by some fella turned into some ranting with Java.

Aside from those, there are other less-popular grumpiness we encouter in daily basis:

Social Media and Public Forums

There are lots of thread out there on which simply asking some question will quickly retaliated with "Let me Google that for you" link - http://lmgtfy.com/ or flood it with ironic/troll answers making the poster feel they had asked very stupid question.

There are actually some popular Twitter account specially crafted to just mock "Clueless Managers" asking some stupid questions/statements
